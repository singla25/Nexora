# 🚀 Nexora – Custom Social Networking Platform (WordPress Engineered) ## 📌 Project Overview Nexora is a fully custom-built social networking platform developed using WordPress (as a base) + Core PHP + AJAX, engineered for scalability, performance, and modular growth. Unlike traditional WordPress solutions, Nexora does not rely on default WP authentication or plugins for core features. Instead, it implements custom systems from scratch, giving full control over: - System architecture - Performance optimization - Data flow & relationships - Feature extensibility --- ## 🎯 Key Highlights - Custom Authentication (No WP default auth) - Fully AJAX-driven architecture - Modular plugin-based system - Custom database design - Chat system built from scratch - Scalable & production-ready structure --- ## 🔐 Advanced Authentication System (Custom Built) - Fully custom login & registration system - Secure session handling (without WP auth) - Input validation & error handling - OTP-based password recovery (email verification) - Auto-login after password reset - Google reCAPTCHA integration for bot protection --- ## 🛡️ Security Enhancement (reCAPTCHA Integration) To enhance platform security and prevent automated attacks, Nexora integrates Google reCAPTCHA into its custom authentication system. ### 🔍 Implemented On: - Login Form - Registration Form ### ⚙️ How It Works: 1. User submits login/register form 2. reCAPTCHA verifies human interaction 3. Token is generated on frontend 4. Token is validated via Google reCAPTCHA API (server-side) 5. Request proceeds only after successful verification ### ✅ Benefits: - Prevents bot-based login attempts - Blocks spam registrations - Protects against brute-force attacks - Strengthens overall authentication security --- ## 🔑 OTP-Based Password Recovery Flow: 1. User submits email / username 2. OTP sent via SMTP 3. OTP verification 4. Password reset 5. Automatic login Future Ready For: - Email verification - Two-Factor Authentication (2FA) --- ## 💬 Chat System (Built From Scratch) ⚠️ Not using any third-party chat plugin ✅ Fully custom-engineered chat system ### 🧠 Core Concept Chat is tightly integrated with Nexora’s connection system: -
